# Employee Management System

A secure and scalable Employee Management System built using Spring Boot and JWT Authentication.

This project demonstrates real-world backend development following clean layered architecture.

## Features
- Employee CRUD operations
- Login & Authentication using JWT
- Attendance tracking
- Salary management
- Role based access
- Exception handling
- RESTful APIs

## Tech Stack
- Java
- Spring Boot
- Spring Security (JWT)
- Spring Data JPA
- Hibernate
- MySQL
- Maven

## Architecture
Controller → Service → Repository → Database

## Modules
- Employee
- Authentication
- Attendance
- Salary

## How to Run
1. Import project into IntelliJ / Eclipse
2. Configure database in application.properties
3. Run EmployeeManagementApplication
4. Use Postman to test APIs

## Purpose
This project is created to demonstrate enterprise backend development skills including security, clean architecture, and REST API design.
